# Privacy Policy for Movie Ratings Sync

**Last updated:** June 29, 2026

## What this extension does

Movie Ratings Sync is a browser extension that helps you export and import movie ratings between Kinopoisk, IMDb, and Letterboxd.

## Data collection

This extension **does not collect, store, or transmit any personal data** to any external servers. All operations are performed locally within your browser.

### What data is processed

- Movie titles, ratings, and watch dates from the pages you visit on Kinopoisk, IMDb, and Letterboxd
- CSV files that you choose to import or export (stored only on your local machine)

### How data is used

- When you click **Export**, the extension reads movie data from the current page and downloads a CSV file to your computer
- When you click **Import**, the extension reads a CSV file that you select and sends the ratings to the respective website's API to set your ratings
- No data is sent to any third-party servers

## Data storage

- Exported CSV files are saved to your computer's Downloads folder
- No data is stored in the extension's local storage
- No data is sent to external analytics services

## Permissions

The extension requests access to Kinopoisk, IMDb, and Letterboxd websites only to read and write movie rating data on those platforms. This data never leaves your browser beyond the interactions with those websites' own APIs.

## Third parties

This extension does not use any third-party analytics, tracking, or advertising services.

## Changes

If this policy changes, you will be notified via the extension's update notes on the store page.
